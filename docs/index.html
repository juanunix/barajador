Ayuda

Aprenda cómo usar el Barajador: simulador de baraja online.

La baraja

Cuando inicie el programa se encontrará con una baraja ordenada de As a K, separada por palos en el órden TriCoPiDia (Tréboles, Corazones, Picas y Diamantes):

Tal como en una extención en cinta la primer carta comenzando por la derecha (A♣) corresponderá a la carta en posición de TOP y la primer carta comenzando por la derecha (K♦) corresponderá a la carta en posisión de BOTTOM.

Posiciones absolutas y relativas

Ajustando las preferencias (o presionando ALT+R) puede activar o desactivar la vista de los números de posiciones sobre cada una de las cartas.

Como notará cada una de la cartas tiene dos números, estos representan las posiciones:

  • Absoluta:
    Arriba del índice. No se modifica con las mezclas. Indica el lugar que se encuentra la carta contando desde Top.

  • Relativa:
    Debajo del índice. Se modifica con las mezclas. Indica dónde se encontraba la carta antes de mezclar.

Como podrá comprobarlo en la baraja de la figura arriba, el 9♠ ocupa la posición 18, habiendo comenzado en la posición 35 contando desde Top.

Matriz

Conceptualmente una baraja de cartas es una matriz unidireccional, una lista de elementos ordenados (cartas virtuales en este caso).

En la figura de arriba se pueden ver tres paquetes distintos de cartas, sus matrices se representan separando cada carta con comas.

Para el full del primer caso KT,KC,KP,8C,8T, para el paquete del centro AC,2C,3C,4C,5C,6C,7C,8C,9C,10C,JC,QC,KC y para el último paquete 6T,2D,6T,JC,5D,9T.

Códigos

Puede utilizar cualquier conjunto de letras o palabras enteras para representar cada carta, pero si quiere que el programa le muestre las imágenes debe seguir la siguiente codificación, la cual de no respetarse dará lugar a una carta en blanco. Los siguientes códigos deben combinarse escribiendo siempre primero el valor seguido por el palo.

Consola

El Barajador cuenta con una terminal de línea de comandos, a la cual puede utilizarse interactivamente escribiendo instrucciones las cuales serán interpretadas: mezclas, ordenaciones, estadísticas e informes.

Sintaxis

Ingrese un comando en la línea que comienza con el símbolo > (el intérprete es indiferente a minúsculas y mayúsculas) y presione la tecla Enter para ejecutar la órden. Puede ingresar varias órdenes utilice ; entre cada una de ellas. Al presionar la tecla Enter se ejecutarán una a continuación de la otra. Ejemplo:

faroExt;invertir
Aplicará a la baraja una mezcla Faro Exterior para luego invertir una a una el órden de las cartas.

Argumentos

Algunos comandos tienen argumentos los cuales modifican la manera en la que el comando se ejecuta. La sintaxis tiene siempre la estructura: comando(argumento1,argumento2). Depediendo el comando los argumentos serán de carácter opcional u obligatorio (en dicho caso al no declararse un argumento válido el comando no se ejecutará).

Multiplicador

Luego de un comando puede utilizar el multiplicador * seguido de un numero entero. Al presionar Enter la órden se ejecutará ese número de veces.

faroExt*6
Aplicará a la baraja seis mezclas Faros Exteriores consecutivas.

Comandos

Esta es una lista de los comando generales con sus descripciones y modos de uso:

version
Imprime la versión del Barajador.
ayuda / help
Muestra un link a esta página.
limpiar / clear
Borra la pantalla de la terminal.
historial / hist
Muestra el historial de comandos ingresados en la terminal.
refrescar / refresh
Recarga la página (no se perderá el estado de la baraja).
generarQr
Genera un código QR a partir con el orden de la baraja actual. Utilizando la API de goQr.me
cortar(n) / cut(n)
Corta un paquete de n cartas de arriba hacia abajo si n es positivo, y de abarjo hacia arriba si nes negativo.
invertir / inv
Invierte una a una el orden de la cartas de la baraja.
invertir(n) / inv(n)
Invierte una a una el orden de las primeras n cartas de la baraja contando desde TOP si n es positivo y desde BOTTOM si n es negativo.
voltear(n) / turn(n)
Voltea la carta en la n posición.
voltear(n,m)
Voltea desde la carta en la n posición m cantidad de cartas. También turn(n,m)
eliminar(n) / del(n)
Elimina la carta ubicada en la n-sima: desde TOP si n es positivo y desde BOTTOM si n es negativo.
voltear(n) / turn(n)
Voltea la carta en la posición n dejándola de dorso si estaba de cara y viceverza.
voltear(n,m)
Voltea la cartas desde la posición n hasta la m, dejándolas de dorso si estaba de cara y viceverza.
orden
Devuelve el órden absoluto de las cartas
reiniciar
Reinicia las posiciones a números contiguos
faroExt / fo
Realiza una mezcla Faro Exterior.
faroInt / fi
Realiza una mezcla Faro Interior.
antiFaroExt / -fo
Realiza una mezcla Antifaro Exterior.
antiFaroInt / -fi
Realiza una mezcla Antifaro Interior.
randomOrg
Mezcla aleatoria a partir del ruido atmosférico, utilizando el algoritmo del servicio Random.Org.
fisherYates
Mezcla pseudoaleatoria utilizando el algoritmo Fisher-yates en su implementación original.
durstenfeld
Mezcla pseudoaleatoria utilizando el algoritmo Fisher-yates en su implementación Durstenfeld.
satollo
Mezcla pseudoaleatoria utilizando el algoritmo Fisher-yates en su implementación Satollo (ninguna carta ocupará la posición original).

Atajos

Para acceder más rapidamente a las distintas opciones del hemos procurado atajos con el teclado para las funciones más comunes.

ALT+C
Abre la ventana de corte.
ALT+O
Mezcla Faro Exterior.
ALT+I
Mezcla Faro Interior.
ALT+L
Mezcla Antifaro Exterior.
ALT+K
Mezcla Antifaro Interior.
ALT+.
Abre la ventana de Mezcla Faro avanzada.
ALT+R
Muestra y oculta las posiciones absolutas y relativas
ALT+Q
Muestra y oculta el módulo de la matriz
ALT+A
Muestra y oculta el módulo del tapete
ALT+Z
Muestra y oculta el módulo de la botonera
ALT+X
Muestra y oculta el módulo de la consola
ALT+G
Abre el graficador
ALT+S
Muestra las estadísticas
ALT+P
Abre las Preferencias de la aplicación
ALT+D
Abre la documentación (está página).
