Ayuda

Aprenda cómo usar el Barajador: simulador de baraja online.

La baraja

Cuando inicie el programa se encontrará con una baraja ordenada de As a K, separada por palos en el órden TriCoPiDia (Tréboles, Corazones, Picas y Diamantes):

Tal como en una extención en cinta la primer carta comenzando por la derecha (A♣) corresponderá a la carta en posición de TOP y la primer carta comenzando por la derecha (A♦) corresponderá a la carta en posisión de BOTTOM.

Consola

El Barajador cuenta con una terminal de línea de comandos, a la cual puede utilizarse interactivamente escribiendo instrucciones las cuales serán interpretadas: mezclas, ordenaciones, estadísticas e informes.

Sintaxis

Ingrese un comando en la línea que comienza con el símbolo > (el intérprete es indiferente a minúsculas y mayúsculas) y presione la tecla Enter para ejecutar la órden. Puede ingresar varias órdenes utilice ; entre cada una de ellas. Al presionar la tecla Enter se ejecutarán una a continuación de la otra. Ejemplo:

faroExt;invertir
Aplicará a la baraja una mezcla Faro Exterior para luego invertir una a una el órden de las cartas.

Multiplicador

Luego de un comando puede utilizar el multiplicador * seguido de un numero entero. Al presionar Enter la órden se ejecutará ese número de veces.

faroExt*6
Aplicará a la baraja seis mezclas Faros Exteriores consecutivas.

Comandos

Esta es una lista de los comando generales con sus descripciones y modos de uso:

version
Imprime la versión del Barajador.
ayuda / help
Muestra un link a esta página.
clear
Borra la pantalla de la terminal.
generarQr
Genera un código QR a partir con el orden de la baraja actual. Utilizando la API de goQr.me
invertir
Invierte una a una el orden de la cartas de la baraja.
faroExt
Realiza una mezcla Faro Exterior.
faroInt
Realiza una mezcla Faro Interior.
antiFaroExt
Realiza una mezcla Antifaro Exterior.
antiFaroInt
Realiza una mezcla Antifaro Interior.
Mezcla aleatoria a partir del ruido atmosférico, utilizando el algoritmo del servicio Random.Org.
fisherYates
Mezcla pseudoaleatoria utilizando utilizando el algoritmo Fisher-yates en su implementación original.
durstenfeld
Mezcla pseudoaleatoria utilizando utilizando el algoritmo Fisher-yates en su implementación Durstenfeld.
satollo
Mezcla pseudoaleatoria utilizando el algoritmo Fisher-yates en su implementación Satollo (ninguna carta ocupará la posición original).
